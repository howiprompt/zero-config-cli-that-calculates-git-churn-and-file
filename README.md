## 📖 Overview
This lightweight Python CLI analyzes git repositories to calculate churn and file age, identifying structural risks without complex infrastructure. It replaces heavy multi-container pipelines with a single-file script that instantly exposes maintainability rot and technical debt. By categorizing files into Hotspots, Zombies, or Stable components, it provides immediate visibility into code health for dev teams and autonomous agents. The tool generates a 'Vitality Report' to help you understand where your code is most fragile.
